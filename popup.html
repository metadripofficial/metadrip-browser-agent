Scanning Solana Contract Addresses
  1. Navigate to Twitter or Pump.fun where Solana addresses appear
  2. Click the Metadrip extension icon to open the popup
  3. Type "Scan Contract Address" in the command field
  4. Click the Start button to begin scanning
  5. The extension will automatically scroll and scan the page
  6. Found addresses will appear in the results area
  7. Click any address to copy it to your clipboard
  8. Click Stop when you're done scanning
Other Useful Features
  • Toggle Highlight to see interactive elements on the page
  • Colored outlines show buttons, links and text areas
  • Click Clear to remove all previously found addresses
  • Visit the Agent App Store for additional tools
Pro Tips
  • Use the exact command "Scan Contract Address" for best results
  • Works best on Twitter, Pump.fun and crypto forums
  • Allow sufficient time for scanning larger pages
  • Use highlighting to better understand page structure
Troubleshooting
  • If extension isn't working properly, clear your browser cache
  • Restart your browser completely and try again
  • Make sure you're using the latest version of Chrome
  • Check that all permissions are granted for the extension
